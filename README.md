# Sentiment Analysis on Financial Documents

### Annotation
ข้อมูลเหล่านี้ถูกกำกับโดยผู้มีพื้นฐานทางด้านเศรษฐศาสตร์ จำนวน 4 คน โดยก่อนการกำกับข้อมูลจริง นักกำกับข้อมูลเหล่านี้ได้ทดสอบกำกับข้อมูลจนสามารถกำกับข้อมูลได้อย่างตรงกัน ด้วยคะแนน Cohen kappa เฉลี่ยระหว่างกลุ่มเท่ากับ 0.73 (Substantial) ในการกำกับข้อมูลทัศนคติ และได้คะแนนเท่ากับ 0.77 (Substantial) ในการกำกับข้อมูลอารมณ์

แม้ว่าลักษณะของชุดข้อมูลที่ตัวบทหนึ่งจะมีการกำกับทัศนคติ หรืออารมณ์ได้มากกว่า 1 ประเภท (Multi-label classification) แต่ผลการทำนายของโมเดล baseline ให้ผลค่อนข้างใกล้เคียงกับการตั้งโจทย์แบบ multi-class ดังนั้น ในการสร้างโมเดลวิเคราะห์อารมณ์และทัศนคติที่อยู่ในเอกสารทางการเงิน จึงทดลองแก้ปัญหาโจทย์แบบ Multi-class classification  หากข้อความใดมีมากกว่า 1 ป้ายกำกับ ป้ายกำกับแรกจะถูกเลือกมาใช้เท่านั้น โดยจะแยกเป็นโมเดลทำนายทัศนคติ 1 โมเดล และโมเดลทำนายขั้วอารมณ์ 1 โมเดล

- โมเดลทำนายทัศนคติ : https://huggingface.co/nlp-chula/augment-sentiment-finnlp-th
- โมเดลทำนายขั้วอารมณ์ : https://huggingface.co/nlp-chula/aspect-finnlp-th

ในการพัฒนาโมเดล คณะวิจัยได้แบ่งชุดข้อมูลออกเป็น 3 ชุด ได้แก่ 
- (70%) Training set จำนวน 8,191 กลุ่มประโยค ระหว่างปี พ.ศ. 2558 -  2561 ของบางบริษัท  
- (15%) Validation set จำนวน 1,756 กลุ่มประโยค  ปี พ.ศ. 2562
- (15%) Test set จำนวน 1,755 กลุ่มประโยค ระหว่างปี พ.ศ. 2561 - 2562 ของบางบริษัท
